CPU Design (Ongoing)

Though an ongoing project, this project is aimed towards designing an 8 bit (upscale to 32 bit) CPU which can run basic tasks.
The whole design is using the universal gate NAND, (NOR can also be used) the first basic designs are being set up on HDL code bits written on online simulators
The design includes:
- ALU (Arithmetic Logic Unit) for basic arithmetic operations
- Control Unit for instruction decoding and execution
- Register File for storing temporary data
